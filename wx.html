file type means
app.json - window backgroundTextStyle 下拉背景字体/loding图的样式，仅支持dark / light
app.json - window navigationBarBackgroundColor 导航栏背景颜色，仅支持HexColor
app.json - window navigationBarTitleText 导航栏标题文字内容
app.json - window navigationBarTextStyle 导航栏标题文字颜色 仅支持black/white
app.json - window enablePullDownRefresh 是否允许下拉刷新，默认不显示需要手动添加，仅支持布尔
app.json - tabbar color 标签栏按钮文字颜色，仅支持HexColor
app.json - tabbar selectedColor 标签栏文字选中时的颜色，仅支持HexColor
app.json - tabbar background 标签栏背景颜色，仅支持HexColor
app.json - tabbar borderStyle 标签栏上方边框颜色，仅支持black/white
app.json - tabbar list 标签栏列表内容，每个值为一个数组，最少两个，最多五个
app.json - tabbar - list pagePath 标签对应的链接地址
app.json - tabbar - list text 标签上每个图标的文字
app.json - tabbar - list iconPath 标签上每个图标图片路径，图片大小40KB 支持png jpg
app.json - tabbar - list selectedIconPath 标签上每个图标选中时的图片路径，图片大小40KB 支持png jpg
app.json - networkTimeout request 设置每次请求有效时间，单位毫秒
app.json - networkTimeout connectSocket 设置链接有效时间，单位毫秒
app.json - networkTimeout uploadFile 设置上传文件有效时间，单位毫秒
app.json - networkTimeout downloadFile 设置下载文件有效时间，单位毫秒
app.js onLaunch 生命周期函数，监听小程序初始化，当小程序初始化完成式，就会触发，全局只触发一次
app.js onShow 生命周期函数，监听小程序显示，小程序启动或者从后台进入前台时触发。
app.js onHide 生命周期函数，监听小程序隐藏，小程序从前台进入后台时触发。
